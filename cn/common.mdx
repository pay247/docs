## 接入说明

* 请求方法Method: POST

* 请求头Header: 
```
"Content-Type":"application/json;charset=UTF-8"
```

* 响应头Header: 
```
"Content-Type":"application/json;charset=UTF-8"
```

* API所使用时区为 ```UTC+8```

## 公共参数

* 公共请求参数

| 参数名       | 类型    | 必选  | 说明     | 示例         |
|:----------|:------|:----|:-------|:-----------|
| mch_id    | 是     | 是   | 商户号 | SVSPYS63XM10042 |
| timestamp | 是     | 是   | 时间戳 | 1594047539 |
| version   | 是     | 是   | 版本 | v2.0 |
| sign      | 是     | 是   | 签名     |b05a16b1e92aa4948e164b747b5749b4 |

* 公共响应参数

| 参数名     | 必须  | 类型      | 说明            | 示例               |
|:--------|-----|:--------|---------------|:-----------------|
| code    | 是   | int     | 状态码，参考字典状态码   | 0                |
| message | 是   | string  | 提示内容/错误信息     | 操作成功             |
| data    | 是   | mixed   | 响应数据          |  响应数据对象或者数组或者null |

## 字典
* 状态码（code）

| 状态码  | 解释   |
|:-----|:-----|
| 0    | 成功   |
| 4000 | 参数错误 | 
| 4001 | 订单重复 |
| 5000 | 系统异常 |
